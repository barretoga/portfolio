## Portfolio
Meu portfolio feito em Vue e inspirado na interface da Steam, juntando as minhas duas paixões que são progamar e jogar.

## 🧑‍💻 Tecnologias
- Swiper.js
- Unplugin components
- Iconify
- Tailwindcss
- Pinia
- Vitest
- Playwright
